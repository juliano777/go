#### Inteiros

Go fornece tipos inteiros com tamanhos fixos, além de tipos cujo tamanho
depende da arquitetura do sistema:  

- **Inteiros de tamanho fixo**  
  - **Com sinal**: `int8`, `int16`, `int32`, `int64`.  
  - **Sem sinal**: `uint8`, `uint16`, `uint32`, `uint64`.  

- **Inteiros dependentes da arquitetura**  
  - `int`: inteiro com sinal padrão, cujo tamanho pode ser 32 ou 64 bits
    conforme a arquitetura do processador.  
  - `uint`: inteiro sem sinal, que corresponde ao `int` no tamanho (32 ou 64
     bits).  
  - `uintptr`: inteiro sem sinal, grande o suficiente para armazenar o valor
     de um ponteiro.  

| Tipo | Tamanho (bits) | Sinal | Intervalo de valores | Uso comum / Aliases |
| :--- | :--- | :--- | :--- | :--- |
| **int8** | 8 | Com Sinal | $-128$ a $127$ | Economia extrema de espaço. |
| **int16** | 16 | Com Sinal | $-32.768$ a $32.767$ | |
| **int32** | 32 | Com Sinal | $-2.147.483.648$ a $2.147.483.647$ | Alias: **`rune`** (usado para pontos de código Unicode). |
| **int64** | 64 | Com Sinal | $\approx -9 \times 10^{18}$ a $\approx 9 \times 10^{18}$ | Necessário para números muito grandes. |
| **uint8** | 8 | Sem Sinal | $0$ a $255$ | Alias: **`byte`** (usado para dados binários). |
| **uint16** | 16 | Sem Sinal | $0$ a $65.535$ | |
| **uint32** | 32 | Sem Sinal | $0$ a $4.294.967.295$ | |
| **uint64** | 64 | Sem Sinal | $\approx 0$ a $\approx 18 \times 10^{18}$ | Necessário para números positivos muito grandes. |
| **int** | 32 ou 64 | Com Sinal | Varia (depende da arquitetura) | **Tipo Padrão.** Usado para a maioria dos inteiros e para índices de *arrays*/*slices*. |
| **uint** | 32 ou 64 | Sem Sinal | Varia (depende da arquitetura) | |
| **uintptr** | 32 ou 64 | Sem Sinal | Varia (depende da arquitetura) | Usado para armazenar o valor numérico de um ponteiro. |
