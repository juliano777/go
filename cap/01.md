# Sobre a linguagem Go

Projeto open source cujo objetivo é fazer programadores serem mais
produtivos.  
Tem como características ser expressiva, concisa, limpa e eficiente.  
Seu mecanismo de concorrência torna fácil escrever aplicações que tirem maior
proveito de multi processamento e máquinas em rede.  
Seu sistema inovador de tipos possibilita criar aplicações flexíveis e 
modulares.  
Go compila de forma rápida para código de máquina, porém oferendo a 
conveniência do recurso de _garbage collection_ em tempo real.  
Linguagem de programação rápida, tipada estaticamente e compilada, que se 
comporta como uma linguagem interpretada e tipada dinamicamente.
